SIARD Suite 2.2 - Composants

SIARD Suite comprend les composants suivants:

  • SiardGui
  • SiardFromDb
  • SiardToDb
  • SiardApi

Haut de page

SiardGui

SiardGui intègre une interface utiliateur graphique interactive permettant le traitement des données en format SIARD. Les métadonnées peuvent être complétées. Les données primaires ne peuvent être modifiées. SiardGui n'est pas approprié pour des recherches complexes. Pour ces dernières, il est recommandé de charger l'archive SIARD dans une base de données et de rechercher à l'aide des techniques de la base de données.

Haut de page

SiardFromDb

L'application lignes de commande SiardFromDb sert à télécharger une base de données et à l'enregistrer dans un fichier SIARD. La fonctionnalité de cette application est identique aux fonctions de téléchargement disponibles dans SiardGui. L'utilisation de la version lignes de commande est particulièrement confortable lors du chargement de grosses bases de données. A l'aide de la redirection de stdout et stderr, le chargement de longue durée peut être aussi mieux documenté (voir „Utilisation des opérateurs de redirection de commande“).

SiardToDb

L'application lignes de commande SiardToDb permet le chargement d'une base de données depuis un fichier SIARD. La fonctionnalité de cette application est identique à celle de chargement disponible dans SiardGui. L'utilisation de la version lignes de commande est particulièrement confortable lors du chargement de grosses bases de données. A l'aide de la redirection de stdout et stderr, le chargement de longue durée peut être aussi mieux documenté (voir „Utilisation des opérateurs de redirection de commande“).

SiardApi

SiardApi est un API JAVA pour la lecture et l'écriture d'archives SIARD. Sa documentation en format Javadoc est disponible dans le répertoire doc/siard-api de distribution de SIARD Suite.

Les fonctions de SiardApi sont toutes intégrées dans siardapi.jar dans le répertoire lib de distribution de SIARD Suite. De plus, les fichiers JAR suivants sont nécessaires:

  • jaxb-api.jar
  • jaxb-core.jar
  • jaxb-impl.jar
  • msv-core-2010.2.jar
  • stax2-api-3.1.1.jar
  • woodstox-core-lgpl-4.1.2.jar
  • woodstox-msv-rng-datatype-20020414.jar
  • xsdlib-2010.1.jar
  • Zip64File.jar
